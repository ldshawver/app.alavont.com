Replit Instructions — Dual-Brand Menu Import + Checkout Conversion + Fulfillment Purge

Build a secure product and ordering system for my app with these rules:

Core concept

Customers browse products under Alavont naming and categories.

At checkout, before payment is created, the app must transform those cart items into Lucifer Cruz branded items for the payment processor.

After successful payment:

the order is sent to the lab technician queue
the receipt/label prints with both names
once fulfillment is complete, the order is deleted or purged from the app database based on admin settings

Do not ask me to confirm basic implementation choices. Build the full feature set in phases and keep the code efficient because Replit usage is expensive.
