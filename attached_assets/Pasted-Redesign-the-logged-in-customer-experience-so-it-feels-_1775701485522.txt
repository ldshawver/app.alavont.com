Redesign the logged-in customer experience so it feels unmistakably like Alavont Therapeutics with a high-energy, electric, polished vibe.

Main goal:
The first screen after login should no longer be a generic dashboard or plain catalog. It should open to an “Order Concierge” AI experience first, with the rest of the ordering flow branching from there.

Overall product direction:
- Brand feel: Alavont Therapeutics
- Mood: electric, energetic, exciting, premium, discreet, modern
- UX should feel alive, not static
- It should still remain fast, readable, and mobile-friendly
- Do not make it childish, cheesy, or cluttered
- Keep it upscale and controlled, with a spark of excitement

Implement these changes:

1. Post-login landing experience
After a user logs in, route them first to the Order Concierge AI screen instead of the generic catalog/home screen.

New first screen should include:
- Alavont Therapeutics branding
- an animated AI concierge character/avatar
- a short first-time welcome/onboarding flow
- clear next steps for ordering

Behavior:
- First login or first visit after account creation:
  show onboarding intro modal/cards
- Returning user:
  go directly to the Concierge screen, but still allow replay of intro

2. Order Concierge AI screen
Create a dedicated hero-style AI concierge experience as the main logged-in landing page.

The concierge should:
- greet the user warmly
- explain the ordering process in simple steps
- explain user responsibility clearly but not aggressively
- help the user start shopping
- answer menu/product questions
- guide user into the order flow

Tone:
- upbeat
- fun
- discreet
- confident
- polished
- high-energy without being spammy

Core intro message requirements:
The first-time explanation should clearly cover:
- how ordering works
- that products are browsed through the secure system
- that checkout/payment is handled through approved channels
- that fulfillment and pickup/delivery status updates happen through the system
- that the user is responsible for reviewing their cart, details, and compliance with any rules shown
- that discretion, accuracy, and responsible use matter

Do not make this read like a legal contract.
It should feel like a concierge orientation, not a scare screen.

3. Concierge character / animation
Create a small animated concierge character or avatar component.

Requirements:
- fun and upbeat
- not creepy
- not childish
- not overly sexualized
- subtle idle animation
- can blink, bob, glow, wave, or pulse
- should visually match the electric Alavont vibe

Suggested implementation:
- animated React component
- use Framer Motion
- floating glow orb / character / assistant badge / stylized bot figure
- lightweight enough for production
- no heavy video requirement unless already optimized

Suggested animation ideas:
- subtle floating
- pulse glow
- electrical shimmer
- animated eye blink
- tiny spark arcs or energy ring
- speaking bounce when bot message appears

4. Visual design language
Redesign the customer-facing logged-in area to feel electric and exciting.

Visual direction:
- dark premium background with vivid electric accents
- neon/electric energy highlights
- smooth gradients
- glassmorphism or luminous card styling where appropriate
- premium sci-fi therapeutic vibe

UI styling requirements:
- hover states should have an electric flash effect
- menu cards should feel energized on hover
- active tabs/buttons should glow
- transitions should feel smooth and intentional
- avoid cheap “casino neon” aesthetics

Implement:
- electric flash hover effect for product/menu cards
- subtle animated energy streak or lightning pulse on hover
- glowing border transition on featured CTA buttons
- energetic but readable category tabs
- strong visual hierarchy

5. Catalog / menu styling
The menu should feel exciting and premium while staying usable.

Requirements:
- product cards feel alive
- hover effect includes:
  - electric edge flicker or flash
  - glow pulse
  - slight lift
- maintain fast performance
- keep text readable
- no excessive animation on low-end devices

Product card ideas:
- dark card base
- colored electric border on hover
- micro spark animation in one corner
- “Order” button glows or energizes on hover

6. First-time user walkthrough
Add a first-time experience overlay or modal carousel.

Show once per user unless manually reopened.

Include 3–5 quick steps such as:
1. Meet your Order Concierge
2. Browse the menu
3. Review your cart carefully
4. Pay through the approved checkout method
5. Track fulfillment and wait for your update

Also include a short responsibility note:
- user must verify order details before paying
- user must follow posted instructions and applicable rules
- user should not share access or misuse the system

Store onboarding completion server-side or in user profile state, not only local storage if possible.

7. Navigation changes
After login:
- default route should be Order Concierge
- catalog/menu still accessible from nav
- orders accessible from nav
- account accessible from nav
- admin/business sitter routes remain role-based and separate

Customer-facing nav should feel branded and premium.

8. Copy requirements
Write better customer-facing copy for:
- first-time welcome
- ordering overview
- responsibility note
- menu intro
- concierge greeting

Example style direction:
- “Welcome to Alavont Therapeutics. I’m your Order Concierge, here to help you explore the menu, build your order, and guide you through the process.”
- “Take a moment to review your selections carefully before checkout. Accuracy, discretion, and responsible use matter here.”

Keep it concise and polished.

9. Accessibility and performance
- preserve keyboard navigation
- do not rely on animation for comprehension
- support reduced motion
- hover effects should degrade gracefully on mobile
- keep animations lightweight
- no blocking intro video

10. Technical implementation
Use:
- React
- Framer Motion for micro-animations
- reusable themed components
- CSS variables or theme tokens for electric style system

Suggested components:
- ConciergeHero
- ConciergeAvatar
- FirstTimeWelcomeModal
- ElectricMenuCard
- ElectricButton
- ElectricHoverBorder
- ResponsibilityNotice
- OrderFlowSteps

11. State and behavior
Track:
- hasSeenConciergeIntro
- canReplayIntro
- firstVisitAfterLogin
- reducedMotion preference

12. Deliverables
Implement:
- new logged-in landing route to Order Concierge
- animated concierge hero section
- first-time onboarding flow
- electric hover effects for menu/catalog
- branded Alavont customer UI polish
- responsibility/process intro copy
- reusable electric design components

13. Do not break
- existing product import
- catalog data loading
- order flow
- auth/session handling
- role-based separation for admin/business sitter

14. QA requirements
Test:
- first login shows onboarding
- returning login skips onboarding and opens concierge
- concierge can still open catalog
- electric hover works on desktop
- mobile remains usable
- reduced motion disables flashy animation
- menu still loads products correctly
- no blank page or render crash from new animations
